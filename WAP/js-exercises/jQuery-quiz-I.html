  1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

    Most jQuery methods are called on jQuery objects as shown below;these methods are said to be part of the $.fn namespace, or the "jQuery prototype," and are best thought of as jQuery object methods.

    $( "h1" ).remove();

    Locate the element(s) we want to modify by using the top-level $ function, which will return them as a jQuery Object which acts like an "array" of DOM objects. This jQuery Object is commonly called the "selection".

    The $ in the jQuery code is just a shorter, more convenient name for the jQuery function.

  2. Write jQuery code to find all h1 elements that are children of a div element and make their background color red.

    Sample HTML:

                        <body>
                        <h1>abc</h1><br><div>
                        <h1>div-1</h1>
                        <h1>div-2</h1>
                        </div>
                        <h1>xyz</h1>
                    </body>
                    

                        $( document ).ready(function() {
                        $("div h1").css("background-color","red");
                    });
